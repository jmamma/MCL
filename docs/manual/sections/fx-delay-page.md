# FX Delay Page

Controls the master FX Delay settings.

![FX Delay page](../assets/images/delay.png)

_To enter the FX Delay Page: hold **[Bank Group]**, then press **[Trig 13]**._

Each parameter of the MD's delay effect can be controlled. Four are displayed at once on screen. Use the FX page-mode toggle to switch between the two parameter groups. On MD-style controls, **[No]** + **[Left]** jumps to the delay page and toggles the current delay group when already there.

## FX A

| Control | Assignment |
| --- | --- |
| Encoder 1 | Modulation (MOD) |
| Encoder 2 | Modulation Frequency (MFQ) |
| Encoder 3 | Mono (MON) |
| Encoder 4 | Level (LEV) |

![FX Delay A parameters](../assets/images/delay_action_2.png)

## FX B

| Control | Assignment |
| --- | --- |
| Encoder 1 | Delay Time (TIM) |
| Encoder 2 | Delay Feedback (FB) |
| Encoder 3 | Filter Frequency (FLF) |
| Encoder 4 | Filter Width (FLW) |

![FX Delay B parameters](../assets/images/delay_action.png)
