# FX Reverb Page

Controls the master FX Reverb settings.

-16pt


![reverb](../assets/images/reverb.png)


-32pt
_To enter the FX Reverb Page: hold **[Bank Group]**, then press **[Trig 14]**._
-16pt

Each parameters of the MD's reverb effect can be controlled. Four of them are displayed at once on screen. Use the FX page-mode toggle to switch between the two parameter groups. On MD-style controls, **[No]** + **[Down]** jumps to the reverb page and toggles the current reverb group when already there.

-4pt
