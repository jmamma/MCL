# Slot Menu

The Slot Menu is used to clear, copy, paste one or more slots and to configure a slot's LEN (length,) LOOP & JUMP settings. It also provides an option to rename the current row and to alternate the active Grid between X or Y.


![slot menu](../assets/images/slot_menu.png)


The Slot Menu is accessible from the GridPage by holding down the MD's **[Exit/No]** key.

When **[Exit/No]** is released, any changes to the menu will be applied to the selected slots.

## Multiple Slot Selection

A rectangular selection containing multiple slots can be made by entering the slot menu (holding **[Exit/No])** and using the MD's **[Up/Down/Left/Right]** arrow keys.
Alternatively rotating MCLs **< Encoder 3>** and **`Encoder 4`**can be used to make a selection.


![range copy](../assets/images/range_copy.png)


## Slot Menu Options

Whilst holding **[Exit/No]** navigate the Slot Menu with the MDs cursor keys, release **[Exit/No]**to confirm the selected action.


| Entry | Function |
| --- | --- |
| Grid: [ X, Y ] | switch between Grid X or Y. |
| Len: (0, 64) | step length of track/slot. |
| Loops: (0, 64) | how many times to loop track/slot. |
| Jump: (0,127) | which row the current slot is to`br` load/jump to after n loops. |
| Clear: [ –, YES ] | clear the selected slot(s). |
| Copy: [ –, YES ] | copy the selected slot(s). |
| Paste: [ –, YES ] | paste the selected slot(s). |
| Rename | rename the current row on Grids X and Y. |
