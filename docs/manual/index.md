# MCL Manual

This manual covers MCL setup, sequencing, grid storage, device configuration and performance workflows.

MCL began on the MegaCommand/MegaCMD hardware platform, with the Machinedrum as the original focus. The project now supports multiple platforms and devices, so this manual uses **MCL** as the project name and refers to MegaCommand only when describing that hardware.

> Back up all projects before installing current firmware. Projects may be upgraded automatically on first load and may not remain compatible with older firmware after conversion.

Recommended device OS versions:

| Device | Required version |
| --- | --- |
| Machinedrum | OS X.13 |
| Monomachine | OS X.01 |

This manual was migrated from the legacy LaTeX/PDF documentation. Some legacy sections and screenshots are still being updated.
