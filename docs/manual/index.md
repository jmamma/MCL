# MCL Manual

| Field | Value |
| --- | --- |
| Firmware version | MCL 5.00 |
| Release date | June 1, 2026 |
| Manual format | Markdown source with GitHub Pages output |

MCL 5.00 introduces a new project format, a reworked MIDI device model, project folders and versions, per-track LFO and arpeggiator storage, expanded sequencer conditions, track fill states, and cross-platform support.

MCL began on the MegaCommand/MegaCMD hardware platform, with the Machinedrum as the original focus. The project now supports multiple platforms and devices, so this manual uses **MCL** as the project name and refers to MegaCommand only when describing that hardware.

> Backup all projects before installing MCL 5.00. Projects are upgraded automatically on first load and are not compatible with older MCL versions after conversion.

Recommended device OS versions:

| Device | Required version |
| --- | --- |
| Machinedrum | OS X.13 |
| Monomachine | OS X.01 |

This manual was migrated from the legacy LaTeX/PDF documentation and is being refreshed section by section for MCL 5.00. Some legacy sections and screenshots are still being updated.
